# 一、深度学习基础

## 1.1 概念

深度学习是一种使用多层神经网络，通过“自动学习特征”来完成任务的机器学习方法。深度学习与经典方法的区别主要在于：前者关注的功能强大的模型，这些模型由神经网络错综复杂的交织在一起，包含层层数据转换，因此被称为*深度学习*（deep learning）。传统算法需要人工设计特征（特征工程），但深度学习**让模型自己从数据中学特征**。这就是深度学习。

## 1.2 组成

深度学习由很多基础组件组成。

### 1.2.1 输入层

- **数据**
    
    数据集的由小变大为现代深度学习的成功奠定基础。仅仅拥有海量的数据是不够的，我们还需要正确的数据。可用数据集通常可以分成两部分：训练数据集用于拟合模型参数，测试数据集用于评估拟合的模型。数据类别可包括：图像（CNN），文本（RNN/Transformer），声音（Speech），数值向量（MLP）等。
    

### 1.2.2 隐藏层

深度学习 = 多层隐藏层。不同类型的网络隐藏层不同：

- MLP（最基本的全连接网络）
    - 每层：线性变换 + 激活函数（ReLU）
- CNN（计算机视觉最常用）
    
    CNN的隐藏层通常包含：
    
    - 卷积层（提取局部特征）
    - 池化层（下采样）
    - 批归一化（稳定训练）
    - 残差连接（ResNet）
- RNN / LSTM（序列处理）
    
    RNN或者LSTM适合处理时间序列、文本。
    
- **Transformer（现在最主流）**
    - Self-Attention（自注意力）
    - FFN（前馈网络）
    - LayerNorm

### 1.2.3 输出层(Output Layer)

深度学习的输出层通常根据任务不同而不同，常见有以下几种：

- 分类任务：softmax 输出概率
- 回归任务：直接输出数值
- 检测人数：边界框 + 类别
- 分割任务：像素级分类

## 1.3 比较

 与传统机器学习方法相比，最直观的一点，就是特征工程，机器学习是人来处理数据，做特征工程，而深度学习是机器自己学习处理数据，自己学习特征。除此之外，深度学习的一个主要优势是可以处理不同长度的数据。机器学习本质上是“从经验中学习”。再而言之，传统 ML：小数据也能训练，但深度学习通常需要大量数据作为学习基础。在小数据表格任务：传统 ML（XGBoost）有时更强，但在复杂任务的时候，深度学习是大幅度领先于机器学习的。深度学习模型巨大，不容易解释。

## 1.4 优化

### 1.4.1 损失函数

深度学习中，大多流行的优化算法通常基于一种基本方法–*梯度下降*（gradient descent）。简而言之，在每个步骤中，梯度下降法都会检查每个参数，看看如果仅对该参数进行少量变动，训练集损失会朝哪个方向移动。 然后，它在可以减少损失的方向上优化参数。贯穿深度学习的总的流程，不难发现，让损失函数最小化（Loss Minimization）是最终目标。对于损失函数而言，不同的任务对应有不同的损失函数：

分类任务：

- **Cross Entropy Loss（交叉熵）**

回归任务：

- **MSE（均方误差）**

检测/分割任务：

- CIoU, BCE, Focal Loss 等

### 1.4.2 梯度计算

模型根据损失函数计算梯度→ 反向传播（BP）算法更新权重，**这是深度学习的数学核心**。

### 1.4.3 优化器（Optimizer）

- SGD + Momentum
    - 加速收敛，有惯性
- Adam（最常用）
    
    结合 SGD 与 RMSProp 的优点：
    
    - 快速收敛
    - 适合大多数任务
- 现代优化器（Transformer 多用）
    - AdamW（加权衰减）
    - LAMB
    - Lion（Meta 开发的新优化器）
